Extensão do prettier que ela ordena as classes do tailwind

https://www.npmjs.com/package/prettier-plugin-tailwindcss

Configuração :

Cria um arquivo .prettierrc na raiz do projeto

Adicionar esse codigo:

{
  "plugins": ["prettier-plugin-tailwindcss"]
}

Reiniciar o VS CODE
